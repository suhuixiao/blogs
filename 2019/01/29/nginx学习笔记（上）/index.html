一. Nginx

Nginx 是一个高性能的 Web 和反向代理服务器

nginx优点：

  • 相比 Apache，Nginx 使用更少的资源，支持更多的并发连接

  • 内存消耗少

  • 安装非常的简单，配置文件 非常简洁

什么是Yum

Yum（全称为 Yellow dog Updater, Modified）是一个在Fedora和RedHat以及CentOS中的Shell前端软件包管理器。基于RPM包管理，能够从指定的服务器自动下载RPM包并且安装

CentOS里使用Yum作为软件包管理工具，yum是linux环境安装软件包的一种方式。
